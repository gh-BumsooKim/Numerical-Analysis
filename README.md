## Roots of Equations
```matlab
% 1. Bisection Method
if func(x_l) * func(x_u) < 0
        x_u = x_r;
else
        x_l = x_r;
end
        
% 2. False-Position Method
x_r = x_u - func(x_u) * (x_l - x_u)/(func(x_l) - func(x_u));

if func(x_l) * func(x_u) < 0
        x_u = x_r;
else
        x_l = x_r;
end

% 3. Fixed Point Iteration
x_new = funcG(x_old);
x_old = x_new;

```

Linear Equation File Usage

```
Ax = b

1*x1  + 0*x2 + 2*x3 + 3*x4 =  1
-1*x1 + 2*x2 + 2*x3 - 3*x4 = -1

6*x1  + 2*x2 + 2*x3 + 4*x4 =  1


-> file_format(.txt) ->


1  0 2  3
-1 2 2 -3
```
